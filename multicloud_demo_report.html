Description
The AWS root account does not have multi-factor authentication (MFA) enabled. The root account has unrestricted access to all AWS resources.
Remediation
Enable MFA for the root account in the AWS IAM console under 'Security credentials'. Use a hardware MFA device for root accounts.

Description

Remediation
